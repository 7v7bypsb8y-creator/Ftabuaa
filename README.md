# FTABUAA LTD — Website

A single-file website for FTABUAA LTD, a building materials store with two branches in Oyarifa, Accra.

## What's in this package

- `index.html` — the entire website (HTML, CSS, and JavaScript in one file). No build tools, no dependencies to install.

## How to publish it

**Fastest option — Netlify Drop (free, no account needed):**
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag `index.html` onto the page
3. You'll get a live link instantly (you can rename it or connect a real domain later)

**Other options:** GitHub Pages, or any standard web host (Hostinger, Bluehost, a `.com.gh` registrar, etc.) — just upload `index.html` as the homepage of your hosting account. No special server requirements; it's plain HTML.

## Editing the content yourself

Everything lives in the one `index.html` file. Open it in any text editor (Notepad, TextEdit, VS Code) and search (Ctrl/Cmd+F) for the text you want to change.

Things you'll likely update over time:

| What | Search for |
|---|---|
| Phone numbers | `244291131` or `540875766` |
| Branch addresses | `Kay Bee Junction` / `Tipa Junction` |
| Opening hours | `hoursTable` (there are two tables, one per branch) |
| Materials/categories | `cat-card` (each product category is one block) |
| Customer quote | search `"One of the leading companies` |

**Important:** phone numbers appear in two places per instance — as visible text (e.g. `024 429 1131`) and as a click-to-call link (e.g. `tel:+233244291131`). If a number changes, update both.

## A few things worth double-checking

These were filled in as reasonable defaults since they weren't listed on the Google Business Profile — confirm and adjust if needed:

- **Hours:** currently set to Mon–Sat, 7:00 AM–6:00 PM, closed Sunday (only the 7 AM opening time was confirmed).
- **Tipa Junction branch:** phone number and hours are currently copied from the Kay Bee Junction branch — update if they differ.
- **Map pins:** both location maps use Google's search-by-name embed. Once the site is live, check that each pin lands on the correct spot and swap in exact coordinates if not.

## After it's live

Consider adding the new URL to your Google Business Profile via "Add website" — this helps customers find you and improves your Google Maps listing.
